# Travel Website

A front-end travel booking website with destination browsing, package details, and booking features.

## Pages

- `homepage.html` - Landing page
- `destinations.html` - Browse destinations
- `packages.html` - View travel packages
- `package-details.html` - Package detail view
- `my-bookings.html` - User bookings
- `login.html` - Login page
- `receipt.html` - Booking receipt

## Tech Stack

- HTML, CSS, JavaScript (Vanilla)

## Getting Started

Just open `homepage.html` in your browser — no build step required.
